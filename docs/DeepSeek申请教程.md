## 申请步骤（5分钟完成）

### 1. 注册账号

访问：https://platform.deepseek.com/

![注册页面](https://platform.deepseek.com)

1. 点击右上角"注册"
2. 使用手机号或邮箱注册
3. 完成验证

### 2. 获取API Key

登录后：

1. 点击右上角头像
2. 选择"API Keys"
3. 点击"创建新的API Key"
4. 复制生成的Key（格式：sk-xxxxxxxx）

⚠️ **重要：请妥善保管API Key，不要泄露给他人！**

### 3. 配置到软件

有两种方式配置：

#### 方式A：首次启动时配置

1. 运行 `start.bat` 或 `python src/main.py`
2. 首次启动会弹出配置向导
3. 选择 DeepSeek
4. 粘贴你的API Key
5. 点击"保存配置并启动"

#### 方式B：手动编辑配置文件

打开 `config/config.yaml`，修改以下内容：

```yaml
ai:
  provider: "openai"  # DeepSeek兼容OpenAI格式
  api_key: "sk-xxxxxxxxxxxxxxx"  # 你的API Key
  model: "deepseek-chat"
  base_url: "https://api.deepseek.com"
```

### 4. 测试连接

配置完成后，运行程序：

```bash
python src/main.py
```

如果看到"✅ AI客户端加载成功"，说明配置正确！

## 常见问题

### Q: API Key在哪里查看？
A: 登录后，点击右上角头像 → API Keys

### Q: 忘记API Key怎么办？
A: 删除旧Key，创建新Key即可

### Q: API调用失败怎么办？
A: 检查：
1. API Key是否正确
2. 网络是否正常
3. 余额是否充足

### Q: 如何查看余额和使用量？
A: 登录平台后，在控制台可以查看

### Q: 可以换成其他AI服务吗？
A: 可以！支持：
- OpenAI GPT (需要国外信用卡)
- Claude (需要国外信用卡)
- 通义千问 (国内服务)

只需修改配置文件即可。
